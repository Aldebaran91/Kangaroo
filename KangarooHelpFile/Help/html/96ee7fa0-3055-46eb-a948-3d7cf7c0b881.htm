What can you do?

The Kangaroo Data Export Management Library standardizes the data transfer of programs to external interfaces. The library makes it possible to make the export dynamic and extensible without requiring major changes to the main system.

A main application case is, for example, a large system conglomerate with many small subsystems that have to transmit information via standardized paths (e.g. email, log files). Since such environments have often evolved historically, there are seldom precisely defined standards. If one of these transmission paths is changed, changes to the individual systems are often necessary. This should be prevented / reduced by the Kangaroo Library with little initial effort. Instead, a system is used that uses standardized export paths that can be exchanged externally without having to change the program.

To keep the initial effort as low as possible, the library provides both a set of useful standard exporters and a set for categorization. However, both can also be adapted by the user. In addition to the exporters, the loading of settings from outside the program is made possible with little effort in order to minimize necessary maintenance requirements on the basic systems.

The Kangaroo Data Export Management Library helps to standardize data transfer and reduce ongoing maintenance.
